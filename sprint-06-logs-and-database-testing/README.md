# Sprint 06 – Logs e Banco de Dados (QA Backend)

## Objetivo
Realizar análise de logs em ambiente Linux e validação de dados em banco PostgreSQL
para identificar erros, inconsistências e padrões de uso.

## Atividades
- Análise de logs com comandos Linux
- Separação de erros HTTP (400 e 500)
- Consultas SQL para validação de dados
- Análise de volume de corridas por empresa

## Ferramentas
- Linux / Bash
- PostgreSQL
- SQL
